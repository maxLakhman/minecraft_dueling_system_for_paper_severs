This plugin is meant for minecraft java edition paper servers

Summary of the plugin: 
It essentially adds a consequenceless PvP system to your server.
You type /duel <playername> to challenge someone to a duel
type /duel accept to accept duel challenges
type /duel help for a summary of these commands
type /duel restore to get your stuff back after a duel

How to put this on your server: 
Clone the repo into IntelliJ or VSCode
Build and run the project, get the jar file and drag it into the "plugins" folder for your server
